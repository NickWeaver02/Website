Main Gamemode: Strike

What is Strike?

Strike is an extraction-style game mode, designed as a replacement for Royalty (old outdated gamemode). The arena is procedurally generated, consisting of large rooms connected by hallways. These rooms can feature random obstacles, and their layout changes each game. The possible room types include: Infirmary Weapons Lab Bio Lab Factory Data Center Holding Cells Docking Bay Headquarters Each round generates a coherent level with these elements arranged randomly.
